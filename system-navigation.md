 
 # get-location 
 this is the command that show you the location of the current directory in the computer using the command lines.
 Demo.
 ## Get-Location # Shows the current folder path
 ## Get-Location | Format-List # Shows detailed path information
 
 # get-childitem
  this is the command to list all the folders or files in the directories.
  ## Get-ChildItem # Lists all files/folders in current folder
 ## Get-ChildItem -Force # Includes hidden files and folders
  
  # set-location
  It is the command for the Shift in the directories like change the directory.
  ## Set-Location Documents # Move to Documents folder
 ## Set-Location .. # Move up one folder level
